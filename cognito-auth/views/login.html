⚡ Humanity Protocol × Cognito

This example shows how apps using AWS Cognito can exchange a Cognito JWT for a Humanity Protocol access token — no second consent screen after the first login.

🔶 Login with AWS Cognito
What happens
1
You log in via the Cognito Hosted UI and your app receives a Cognito id_token.
2
First time only: you grant the Humanity consent screen to link your Humanity account.
3
Every subsequent login: the backend exchanges the Cognito JWT for a Humanity access token in a single server-to-server call — no UI interaction needed.
4
Use the HP token to verify presets, fetch the user's profile, and gate access to features.
